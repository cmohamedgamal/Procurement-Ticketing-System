Connection Settings
1
Find your Fusion URL
Log into Oracle Fusion Cloud. Copy the base URL from your browser address bar — stop before /fscmUI.
https://[company]-test.fa.us2.oraclecloud.com
2
Create Integration User
In Fusion, go to Security Console → Users. Create a user with the roles: Procurement Integration Specialist and Supplier Portal Manager.
3
Enable REST Services
Confirm REST services are enabled via: Setup & Maintenance → Offerings → Procurement → Feature Selection. Enable REST Inbound Suppliers.
Navigator → Tools → REST API Explorer
4
Test & Go Live
Click "Test Connection" to verify. Once connected, ProcureDesk will pull Suppliers, POs, and Contracts from Fusion automatically.
Oracle Fusion REST Endpoints Used
GET/fscmRestApi/resources/11.13.18.05/suppliersPull supplier list
GET/fscmRestApi/resources/11.13.18.05/purchaseOrdersPull POs
GET/fscmRestApi/resources/11.13.18.05/purchaseOrders/{id}/linesPO line details
POST/fscmRestApi/resources/11.13.18.05/suppliersCreate supplier
GET/fscmRestApi/resources/11.13.18.05/supplierInvoicesInvoice disputes
PATCH/fscmRestApi/resources/11.13.18.05/purchaseOrders/{id}Update PO status
GET/fscmRestApi/resources/11.13.18.05/contractHeadersPull contracts
ℹ This tool runs entirely in your browser. Credentials are stored only in your browser's localStorage and are never sent to any third-party server. All API calls go directly from your browser to your Oracle Fusion instance.
